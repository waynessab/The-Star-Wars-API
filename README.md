# About the project
Project developed to be an application to collect information from characters, planets and starships of "The Star Wars API" (https://swapi.dev/)

The project consists of 3 APIÂ´s that search the site data (https://swapi.dev/) and are able to list the information of a person, planet and random ship. And another API able to list the names of all characters at random.
